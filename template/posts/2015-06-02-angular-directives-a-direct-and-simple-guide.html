Angular Directives: a direct and simple guide

JUN 02, 2015

What is a Directive?

A directive is a marker that you will add to your HTML code to attach a specific behavior. This marker can be an element, attribute, class or comment. Let's try this with an element mark. If you need to create an accordion for your Angular app, you could do the following:

<accordion>
    <div>My collapsible content</div>
</accordion>

The <accordion> element is your directive. The next step is to add your desired behavior. In this case, we'll add a toggle button and show/hide the element when we click on it.

angular.module('myApp', [])
    .directive('accordion', function() {
        return {
            transclude: true,
            template: '<input type="button" value="Toggle"><ng-transclude/>',
            link: function(scope, elem, attrs) {
                elem.find('input').bind('click', function() {
                    elem.find('ng-transclude').toggleClass('hidden');
                });
            }
        };
    });

In this example, we used the following elements inside the directive:

  • transclude: true - this tells Angular that you want to use the content inside your directive inside your your directive template.
  • template - this is the HTML content that you will append inside your directive. In this case, we added an input button and the <ng-transclude/> element that marks where the original content will be placed.
  • link - this is where you will add your directive behavior. You can use jqLite that is a subset of jQuery. It has less features than jQuery to enforce better practices. For example, you should not manipulate DOM elements that are out of your directive scope.
